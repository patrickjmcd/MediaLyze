# Contributing

## Development

1. Create a virtual environment and install backend dependencies.
2. Install frontend dependencies in `frontend/`.
3. Run the FastAPI app and the Vite dev server in parallel while developing UI changes.

## Expectations

- Keep modules focused and typed.
- Add or update tests for changed logic.
- Preserve read-only behavior for media files.
- Document config or behavior changes in `README.md`.

## Pull Requests

- Keep scope focused.
- Describe user-visible behavior changes.
- Include local verification details.
- Do not include secrets or machine-specific artifacts.
